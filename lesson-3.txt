1. Giải thích các câu lệnh: git commit --amend, git commit --amend
-m”<message>”, git restore --staged <file>, git reset HEAD~1.

- Answer -
git commit --amend: thay đổi commit gần nhất và message của commit đó bằng cách mở ra text editor
git commit --amend-m”<message>”: thay đổi message của commit bằng cách thay thế <message> vào lệnh luôn chứ không cần text editor
git restore --staged <file>: đưa trả <file> từ vùng Staging Area về vùng Working Directory
git reset HEAD~1: đưa tất cả những thay đổi trong commit gần nhất từ vùng Repository về lại vùng Working Directory

2. Hãy tưởng tượng có 1 folder lesson-3 mới được tạo. Hãy liệt kê các file theo từng
vùng, liệt kê các commit và message tương ứng sau khi thao tác một loạt các lệnh sau
(sẽ rất tuyệt vời nếu bạn không chạy trên máy tính thật, mà bạn chạy trong đầu, sẽ giúp
bạn hiểu về Git rõ ràng hơn):
a. Tạo 3 file: file1, file2, file3
b. Chạy lệnh: git init
c. Chạy lệnh: git commit -m”init project”
d. Chạy lệnh: git add file1
e. Chạy lệnh: git commit -m”add file”
f. Chạy lệnh: git add .
g. Chạy lệnh: git restore --staged file2
h. Chạy lệnh: git commit -m”add all files”
i. Chạy lệnh: git reset HEAD~1

- Answer -
a. Tạo 3 file: file1, file2, file3
- Local: file1, file2, file3

b. Chạy lệnh: git init
- Working Directory: file1, file2, file3
- Staging Area:
- Local Repository:

c. Chạy lệnh: git commit -m”init project”
- Working Directory: file1, file2, file3
- Staging Area:
- Local Repository:
    - Commit: nothing to commit

d. Chạy lệnh: git add file1
- Working Directory: file2, file3
- Staging Area: file1
- Local Repository:

e. Chạy lệnh: git commit -m”add file”
- Working Directory: file2, file3
- Staging Area:
- Local Repository: file1
    - Commit C1: file1
    - Message: add file

f. Chạy lệnh: git add .
- Working Directory:
- Staging Area: file2, file3
- Local Repository: file1

g. Chạy lệnh: git restore --staged file2
- Working Directory: file2
- Staging Area: file3
- Local Repository: file1

h. Chạy lệnh: git commit -m”add all files”
- Working Directory: file2
- Staging Area: 
- Local Repository: file1, file3
    - Commit C2: file3
    - Message: add all files

i. Chạy lệnh: git reset HEAD~1
- Working Directory: file2, file3
- Staging Area:
- Local Repository: file1
